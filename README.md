### 서버 실행 방법
* 이 리포지트리 클론
```bash
git clone https://github.com/notdevblue/OneTwoShotgun.git
cd OneTwoShotGun/src/server
```
<br/>

* 모듈 다운로드 
```bash
npm install
```
<br/>

* (윈도우인 경우) server/.env 파일에 PWD 를 추가
```bash
PWD=(경로)/OneTwoShotgun/src/server
```
<br/>

* 서버 실행
```bash
node index.js
```
또는
```bash
nodemon
```
